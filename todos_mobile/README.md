## Todo Model:
|      Attribute   |     Type    |  
|------------------|-------------|
|        title     |    STRING   |
|     description  |     TEXT    |
|       isDone     |     BOOL    |
|       remind     |   DATETIME  |
|    daysToRemind  |   BIT ARRAY |
